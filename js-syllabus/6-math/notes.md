## Math Object

### What is Math object?
- Math js me ek built-in object hai jisme kuch methods hai jo hamen mathematical operation me help krte hai.

#### Methods of Math - 
| Method              | Use                       | Example             | Output        |
| ------------------- | ------------------------- | ------------------- | ------------- |
| `Math.round(x)`     | Nearest integer tak round | `Math.round(4.6)`   | `5`           |
| `Math.floor(x)`     | Niche integer             | `Math.floor(4.9)`   | `4`           |
| `Math.ceil(x)`      | Upar integer              | `Math.ceil(4.1)`    | `5`           |
| `Math.trunc(x)`     | Decimal part remove       | `Math.trunc(4.9)`   | `4`           |
| `Math.random()`     | 0-1 random number         | `Math.random()`     | `0.3478`      |
| `Math.max(a,b,...)` | Max number                | `Math.max(10,5,20)` | `20`          |
| `Math.min(a,b,...)` | Min number                | `Math.min(10,5,20)` | `5`           |
| `Math.abs(x)`       | Absolute value            | `Math.abs(-7)`      | `7`           |
| `Math.sqrt(x)`      | Square root               | `Math.sqrt(16)`     | `4`           |
| `Math.pow(x,y)`     | Power x^y                 | `Math.pow(2,3)`     | `8`           |
| `Math.PI`           | Pi ka value               | `Math.PI`           | `3.141592...` |

### Why Math Created?
- Math object se hame numbers pe kuch commonly used operation ke liye methods mil jate hai. Nhi to hame khud se banana padta. That's the reason behind it.

### How Math works ?
- Math directly static methods provide karta hai, isliye object ko create nahi karna padta.
